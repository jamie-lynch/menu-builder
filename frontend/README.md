## Running Locally

**Requirements**

-   Node (v14.3.0)
-   Yarn

**Setup**

-   Install dependencies - `yarn`
-   Create a `.env.` file based on the `.env.example` file with the env vars for running locally

**Running**

-   Start the app - `yarn start`
